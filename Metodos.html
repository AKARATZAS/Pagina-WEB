Métodos de cálculo de intercambiadores de calor

A la hora de diseñar los intercambiadores de calor existen varios métodos aplicables que ayudarán al ingeniero a conseguir los resultados de una manera más sencilla y eficaz posible. De esta forma se podrá conseguir encontrar el intercambiador de calor óptimo para cada cliente.


Método epsilon-NTU

El método que os introduciremos es el método de espilon-NTU, el cual siver para resolver problemas de rendimiento. El fundamento de este método se basa en el cálculo de dos números adimensionales, epsilon y NTU de los cuales podremos graficarlas de manera que se podrá determinar un tercer número adimensional con el que se podrán finalmente calcular las temperaturas de salida.

Desarrollado por London y Seban en 1942, este método es muy usado hoy en día cuando se quiere seleccionar un intercambiador entre varios disponibles o si se necesita para un uso diferente para el que se diseñó en su día.

Parámetros:

mh: Flujo másico del fluido caliente

mc: Flujo másico del fluido frio

Th,in: Temperatura de entrada del fluido caliente

Th,out: Temperatura de salida del fluido caliente

Tc,in: Temperatura de entrada del fuido frio

Tc,out: Temperatura de salida del fluido frio

Cp,h: Calor específico del fluido caliente

Cp,c: Calor específico del fluido frio

Ch: Capacidad de transferencia del fluido caliente, Ch=mh*Cp,h

Cc: Capacidad de transferencia del fluido frio, Cc=mc*Cp,c


  1. qmax: El máximo q posible. Cuando la diferencia de temperaturas es máxima para uno de los flujos. Esto puede ser calculado si el intercambiador de calor tiene una longitud infinita.

    2. q max=Cmin*ΔTmax

    ΔTmax= Th,in-Tc,in


  2. Efectividad, ε: Relación entre el actual q con qmax

    3. ε=q/qmax=Cc*(Tc,out-Tc,in)/Cmin*(Th,in-Tc,in) =Ch*(Th,in-Th,out)/Cmin*(Th,in-Tc,in)


  3. Ratio de capacidad de transferencia, Cr:

    4. Cr=Cmin/Cmax
    Cambio de fase: Cmax→ ∞ , entonces Cr→ 0


  4. Número de unidades de transferencia, NTU

    5. NTU= U*A/Cmin
